# 🪑 欢迎来到「圆桌 RoundTable」

> 你问一个问题,多个 AI 模型围着圆桌各自作答、参考他人回答后修正自己,最后由「主席」打分排序、合并出一份更全面的最终答案。

## 一轮辩论的 4 个阶段

1. **各自作答** —— N 位成员并发各答一次
2. **各自修正** —— 每位成员看到他人(匿名)的初始回答,参考后修正自己
3. **主席打分** —— 主席给每个修正回答打分并排序
4. **主席合并** —— 主席把所有修正回答合并成最终答案

等待期间,界面会实时显示进度走到哪一步,不会全程黑盒;辩论过程默认折叠在每条回答下,点开即可查看各成员的原始回答、修正回答与主席评分表。

## 怎么开始

1. **配置模型**:首次使用先复制模板并填好你的 API key:
   ```bash
   cp models.example.yaml models.yaml
   ```
   然后编辑 `models.yaml`,把每条 `api_key:` 占位符换成真实 key。加减选手 = 增删 `model_list` 条目。
2. **启动**:`conda activate roundtable && chainlit run app.py`(或 `source run.sh`)。
3. **提问**:在下方输入框直接问即可,每轮约 30~90 秒。

> 若有 key 没填或占位符还在,界面会明确提示缺哪个,补好重启即可。
